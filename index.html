Prompt Masters

How to Play

  • Join a Room: Enter a room ID to create or join a room.
  • Start the Game: Start when enough players (4+) have joined.
  • Generate Images: Receive prompts, generate images, and submit your favorite.
  • Vote on Images: Vote for the best submitted image in the pair.
  • Scoring: Earn points for submitting the winning image in your pair.
  • End of Game: After the set rounds, the highest scorer wins.
